## Screenshot Best Practices

1. **Resolution:** 1920x1080px minimum for clarity
2. **Format:** PNG (better quality) or WebP (smaller size)
3. **Optimization:** Use tools like TinyPNG to reduce file size
4. **Consistency:** Same theme, same browser window size
5. **Privacy:** No personal data, API keys, or sensitive information
6. **Branding:** Clean UI, no development tools visible

## Tools for Screenshot Capture

- **Windows:** Win + Shift + S (Snipping Tool)
- **macOS:** Cmd + Shift + 4
- **Chrome DevTools:** Cmd/Ctrl + Shift + P → "Capture screenshot"
- **Firefox:** Cmd/Ctrl + Shift + S
